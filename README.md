3. 本库 C++ 代码缺省源解释：

   ```c++
   #include<bits/stdc++.h>//万能头文件，平时练习懒得写那么多头文件可以用
   #define int long long//强制全局的整型变量均为长整型，防止数据爆 int
   using namespace std;
   signed main(){//由于前面 define 了int，所以为了主函数正常进行使用了 signed
       
       return 0;
   }
   ```

4. Python 代码若缺少对应第三方库可以在终端使用如下命令

   ```shell
   pip install 第三方库名
   ```
