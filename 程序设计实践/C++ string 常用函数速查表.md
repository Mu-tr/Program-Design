### 📋 C++ `std::string` 常用函数速查表

| 功能分类     | 函数 / 语法              | 说明                                          |
| ------------ | ------------------------ | --------------------------------------------- |
| **基础属性** | `size()` / `length()`    | 返回字符数量                                  |
|              | `empty()`                | 判断是否为空                                  |
|              | `c_str()`                | 返回 C 风格字符串 `const char*`               |
| **访问元素** | `str[i]`                 | 访问第 i 个字符（无边界检查）                 |
|              | `str.at(i)`              | 访问第 i 个字符（**有边界检查，越界抛异常**） |
|              | `front()` / `back()`     | 获取首/尾字符                                 |
| **修改操作** | `str += s` / `append(s)` | 末尾追加字符串                                |
|              | `push_back(c)`           | 末尾追加**单个**字符                          |
|              | `pop_back()`             | 删除末尾**单个**字符                          |
|              | `clear()`                | 清空字符串                                    |
| **插入操作** | `insert(pos, s)`         | 在 `pos` 处插入字符串 `s`                     |
|              | `insert(pos, n, c)`      | 在 `pos` 处插入 `n` 个字符 `c`                |
| **删除操作** | `erase(pos, len)`        | 从 `pos` 开始删除 `len` 个字符                |
| **替换操作** | `replace(pos, len, s)`   | 从 `pos` 开始，将 `len` 个字符替换为 `s`      |
| **查找操作** | `find(s, pos)`           | 正向查找，返回首次出现位置，失败返回 `npos`   |
|              | `rfind(s)`               | 反向查找，返回最后一次出现位置                |
| **截取操作** | `substr(pos, len)`       | 截取从 `pos` 开始、长度为 `len` 的子串        |
| **数值转换** | `to_string(val)`         | 数值 ➡️ 字符串                                 |
|              | `stoi(s)` / `stod(s)`    | 字符串 ➡️ `int` / `double`                     |
| **比较操作** | `==`, `!=`, `<`, `>`     | 字典序比较（最常用）                          |
|              | `compare(s)`             | 返回 0(相等), <0(小于), >0(大于)              |
